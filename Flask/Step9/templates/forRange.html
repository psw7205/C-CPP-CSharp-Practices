for..range

1~10까지 출력하기

    {% for i in range(1,11) %}
  • {{i}}
    • {% endfor %}

5단의 구구단을 출력하여라

    {% for i in range(1,10) %}
  • 5 X {{i}} = {{5*i}}
    • {% endfor %}

구구단 모두를 출력하여라

{% for i in range(2,10) %}

{{i}} 단 출력하기

    {% for j in range(1,10) %}
  • {{i}} X {{j}} = {{i*j}}
    • {% endfor %}
{% endfor %}
